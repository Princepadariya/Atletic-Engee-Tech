# Install Node.js

To run this project, you need Node.js and npm installed on your machine.

## Recommended Version
We recommend using the latest **LTS** (Long Term Support) version of Node.js.

## Download
Visit the official Node.js website to download the installer:
[https://nodejs.org/en/download/](https://nodejs.org/en/download/)

## Verify Installation
After installing, open your terminal and run:
```bash
node -v
npm -v
```

## IDE Recommendation
We recommend using **Visual Studio Code** for the best development experience.
